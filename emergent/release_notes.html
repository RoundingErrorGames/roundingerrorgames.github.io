Release notes for Deciduous: Emergent.

Deciduous is in development and not yet available to the public. These release notes track changes in our internal and closed test tracks. If you are interested in participating as a beta tester, please contact support@roundingerrorgames.com.

Deciduous will launch in the Android Play Store in Summer 2025.


v0.1.24 version code 22 released 2025-05-05
  • Heavy revision of struct Cell and struct Evolve.
  • REPLAY may complain in logs about legacy files, but still works.
  • Less-spooky random numbers: each time and place has its own sequence.
  • Cut genes for cell movement direction.
  • Increased granularity of gene for radiation resistance.
  • Added gene for radiation mutability.
