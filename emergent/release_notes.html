Release notes for Deciduous: Emergent.

Deciduous is in development and not yet available to the public. These release notes track changes in our internal and closed test tracks. If you are interested in participating as a beta tester, please contact support@roundingerrorgames.com.

Deciduous will launch in the Android Play Store in Summer 2025.


v0.1.13 version code 9 released 2025-03-27
  • Small font for copyright text, added the arch back in.
  • Time travel is back! Via clicking on the metric graph. Warning: the old future is lost!
  • Modals build up in a "stack": an originating modal remains visible behind any spawned confirmation or error modals.
  • Added some debug beachhead to the "dead zone", the center of the center bar.
  • Bump targetSdkVersion from 34 (Android 14) to 35 (Android 16).

  • Removed arch from copyright text for screen size. The arch is in DEBUG.
